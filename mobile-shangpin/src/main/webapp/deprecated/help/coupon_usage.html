优惠券使用

什么是优惠券

优惠券是尚品网为了回馈用户，发到用户账户中，可抵同面值金额的券。

每张优惠券均有它的使用条件，请在遵循优惠券的规定条件下使用（特殊商品除外）。

如何获取优惠券

尚品网会不定期的举行一些回馈活动，只要积极参与活动，就有机会获得优惠券。

如何使用优惠券

当您将商品加入购物车，去结算时，可以使用符合条件的优惠券。

详见优惠券使用流程。

如何查看优惠券

进入"我的账户"，在交易管理中，可点击查看"我的优惠券"。

优惠券使用规则

1. 每笔订单只能使用一张优惠券，每张优惠券只能使用一次，且不能用于购买或兑换充值礼品卡。

2. 某些特殊商品不能使用优惠券。

3. 优惠券需在有效期内使用，过期则不能再使用。

4. 优惠券不予兑现，如遇退单情况，退款仅按照实际支付金额结算。

优惠券使用流程

  • Step.1选择商品加入购物袋，点击"去结算"，进入"填写核对订单信息"页面。
  • Step.2在"填写核对订单信息"页面左下方选择您要使用的优惠券。
  • Step.3点击"确认订单，去支付"。
