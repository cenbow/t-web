会员级别共分四级，具体为：普通会员、黄金会员、白金会员、钻石会员，级别升降均由系统自动实现，无需申请

一、会员升级制度及会员权利：

普通会员

  • 升级条件：

    凡注册尚品的用户，即可成为尚品网的普通会员

  • 会员权利：

    1. 可以享受普通会员所能购买的商品及相关服务
    2. 可获赠100元生日红包优惠券

黄金会员

  • 升级条件：

    一年内购物满5000元即可升级为黄金会员，有效期为一年。

  • 会员权利：

    1. 可以享受全场购物最低9.5折的优惠
    2. 可获赠100元生日红包优惠券
    3. 可获赠300元升级红包优惠券
    4. 国内订单消费金额满399元免运费

白金会员

  • 升级条件：

    一年内购物满10000元即可升级为白金会员，有效期为一年。

  • 会员权利：

    1. 可以享受全场购物最低9.0折的优惠
    2. 可获赠100元生日红包优惠券
    3. 可获赠500元升级红包优惠券
    4. 国内订单消费金额满299元免运费

钻石会员

  • 升级条件：

    一年内购物满30000元即可升级为钻石会员，有效期为一年。

  • 会员权利：

    1. 可以享受全场购物最低8.5折的优惠
    2. 可获赠100元生日红包优惠券
    3. 可获赠1000元升级红包优惠券
    4. 国内订单消费金额满199元免运费

二、会员保级制度：

  • 黄金会员：

    一年内消费满5000元可保持黄金会员级别。不满5000元将降级为普通会员。

  • 白金会员：

    一年内消费满10000元可保持为白金会员级别。不满10000元将降级为黄金会员。

  • 钻石会员：

    一年内消费满30000元可保持为钻石会员级别。不满30000元将降级为白金会员。

三、等级生效：

1. 会员等级的升降，系统会自动进行。

2. 一年内购物满5000元、10000元、30000元指的是从当天的凌晨开始计算前366天内的交易全部完成订单的购物总额。

四、生日红包和升级红包的使用规则：

1. 生日红包使用规则：

生日红包有效期均为30天，从发放之日算起。红包尚品网全站可用，但尚品网旗下网站“尚品奥莱网”及指定特殊品牌、特殊商品不可享受。其他要求：生日红包要求必须准确填写出生日期（完善出生日期），一年仅赠送一次。

2. 升级红包使用规则：

升级红包在用户每次等级向上变更时赠送，发放日期为等级变更后当日

升级为黄金会员时，发送300元升级红包。分成两张优惠券：100元、200元，每个订单只能使用一张优惠券；

升级为白金会员时，发送500元升级红包。分成三张优惠券： 100元、200元、200元，每个订单只能使用一张优惠券；

升级为钻石会员时，发送1000元升级红包。分成四张优惠券： 100元、200元、200元、500元，每个订单只能使用一张优惠券；

特殊说明：

1. 累计消费金额：指用户实际支付金额（包含尚品网和尚品奥莱网实际支付金额），不包括购物折扣费用、现金红包费用、优惠券费用等；

2. 购物退货，累计消费金额按实际退款费用扣减，对于所使用的红包优惠券不退还；

3. 购买折扣：尚品网全站享受，但尚品网旗下网站“尚品奥莱网”及指定特殊品牌、特殊商品不可享受；

4. 免费送货上门是指全场商品提供免费送货上门服务，不收取任何快递/物流费用；

5. 黄金会员、白金会员、钻石会员连续保持两年会员级别则第三年最低可以保持该会员级别；

6. 所有会员需要支持尚品网，维护尚品网合法权益及公司形象，禁止恶意攻击或诽谤；

7. 以上会员升降级及相关权益最终解释权归尚品网所有，如有疑问请致电客服电话4006-900-900咨询。
